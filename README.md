#  MERN Banking Management System

A full-stack banking web application built using the MERN stack that allows users to securely manage their accounts, transfer money, withdraw funds, and receive transaction notifications. The application provides authentication, transaction tracking, and a responsive user interface.

---

##  Features

* Secure user authentication using JWT
* Account balance management
* Money withdrawal functionality
* Secure money transfer between accounts
* Transaction history tracking
* Real-time transaction notifications
* Responsive and modern user interface

---

##  Tech Stack

**Frontend**

* React
* Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB
* Mongoose

**Authentication**

* JSON Web Token (JWT)

##  Installation

### Clone the repository

```
git clone https://github.com/abinavedavana/Banking-Management-System.git
```

### Navigate to the project directory

```
cd Banking-Management-System
```

### Install backend dependencies

```
cd backend
npm install
```

### Install frontend dependencies

```
cd ../frontend
npm install
```

### Run the application

Backend:

```
npm run dev
```

Frontend:

```
npm run dev
```
